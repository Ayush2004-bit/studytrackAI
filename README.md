# 🎓 StudyTrack AI

An AI-powered Student Performance Analysis and Recommendation System built using **Flask**, **Machine Learning (K-Means Clustering)**, and **Python**. The application helps analyze students' academic performance, study habits, attendance, stress levels, and lifestyle to generate personalized insights and recommendations.

---

## 🚀 Features

- 🔐 Admin Login System
- 📊 Student Performance Dashboard
- 👨‍🎓 Student Profile Analysis
- 📈 Performance Analytics
- 🧠 Machine Learning-based Student Clustering (K-Means)
- 💬 AI Study Assistant Chat
- 📚 Personalized Study Recommendations
- ⚖️ Student Comparison Module
- 😴 Lifestyle & Study Habit Analysis
- 📉 Cluster-wise Performance Insights

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Ayush2004-bit/studytrackAI.git
```

### Move into Project

```bash
cd studytrackAI
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
